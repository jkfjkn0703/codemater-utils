# Java Utilities Library

A lightweight, practical Java utility library providing common helper functions for string manipulation, mathematical calculations, and file operations.

## Features
- **StringUtils**: String utility methods (isBlank, capitalize, join, etc.)
- **MathUtils**: Mathematical utilities (prime check, GCD calculator, etc.)
- **FileUtils**: File I/O operations (read, write, copy)
